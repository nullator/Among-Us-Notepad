# Among Us Notepad (Among Us мод) — локальный блокнот для заметок по игрокам Among Us

Этот мод добавляет в Among Us интерфейс для просмотра данных об игроках, вместе с которыми вы играли в одном лобби.
Он работает **локально на вашем компьютере** и поднимает встроенный сервер на `http://localhost:33333/`.
Все данные хранятся в папке `BepInEx\config\Notepad`.

## Что это за архив и для какой ОС
- Мод предназначен для **Windows** (Among Us под Windows).

## Требования
- Установленная игра **Among Us** (Windows версия).
- Установленный **BepInEx для IL2CPP** (под Among Us).

## Как запускать после установки
1. Запускайте игру **как обычно** (через Steam/ярлык).
2. При успешном запуске мод открывает вкладку браузера на `http://localhost:33333/`.
3. Внутри игрового лобби в правом верхнем углу появится кнопка, которая будет открывать или скрывать окно с заметками по игрокам лобби и с данными о предыдущих никах.
4. Вы можете добавить любому игроку личную пометку через окно в браузере. Эта пометка **сохранится** и будет показываться в игровом лобби каждый раз, когда этот игрок будет заходить в лобби (неважно, под каким ником).

# Установка (пошагово)

## 0) Если ранее вы уже устанавливали другие моды с BepInEx
Пункты 1 и 2 можно пропустить. Возможно, потребуется обновить BepInEx, если мод не запустится.

## 1) Скачать и установить BepInEx (IL2CPP)
1. Скачайте последнюю версию BepInEx **для IL2CPP** (под Among Us) со страницы https://github.com/BepInEx/BepInEx/releases.
2. Распакуйте содержимое архива **в корень игры**, чтобы появились файлы/папки:
   - `BepInEx\`
   - `doorstop_config.ini`
   - `winhttp.dll`

## 2) Первый запуск игры (создание папок)
1. Запустите игру **один раз**.
2. BepInEx выполнит первоначальную инициализацию, скачает необходимые файлы и создаст новые директории.
3. Дождитесь главного меню (это может занять много времени!) и выйдите.
4. Убедитесь, что появились папки:
   - `BepInEx\plugins`
   - `BepInEx\config`

## 3) Установка мода
1. Распакуйте архив с модом в папку **BepInEx**.
2. В результате должно получиться, что:
   - файл `Notepad.dll` находится в папке `BepInEx\plugins`
   - папка `Notepad` находится в папке `BepInEx\plugins\Notepad\`

То есть Итоговая структура должна быть такой:
`путь до папки с игрой\Among Us\BepInEx\plugins\Notepad.dll`
`путь до папки с игрой\Among Us\BepInEx\plugins\Notepad\(тут необходимые для работы мода файла)`

## 4) Первый запуск мода
1. Запустите игру.
2. Мод автоматически откроет браузер на `http://localhost:33333/`.

Если вкладка браузера не открылась автоматически, откройте вручную:
`http://localhost:33333/`
